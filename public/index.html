MBTA departure board challenge.

You'll need a valid apiKey for streaming predictions to work. Can be a query string parameter, or hard-coded in datalayer.js. Errors logged to console.

https://rmela.github.io/mbta_departure_board/public?apiKey=your_api_key_here

Challenge requirements:

  • Show upcoming departures at North and South stations
  • Show departure track id**
  • Show train destination
  • Show departure time
  • Show boarding status

**While I could get departure track using includes when fetching predictions, I had trouble figuring out how to do it for streamed predicitons. And I'm out of time.
